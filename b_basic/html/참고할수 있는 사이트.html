강조에 대한 표현

P는 문단의 표현

하나의 문단에서 줄바꿈을 처리하기 위해서는 현위치를 기준으로
줄바꿈처리하는 요소를 삽입

내용에 대한 설명을 작성하는 경우, 대에따라 강조의 표현은 해야하는 경우가 발생한다.

내용의 강조의 표현 중에서 위험, 중대한,긴급의 강조의 표현
내용상의 강세의 표현
새로 추가된 흐름의 표현
삭제된 표현
내용상의 강조를 표현하는 형태로 그 의미롸 기능을 명확하게

글씨의 굵기 : bold - 요약 키워드, 단순 굵기표현
글씨의 기울기,외국어 등장인물의 생각, 생각의 표현(아이콘,이모티콘)
밑줄

block 요소, inline 요소

block 요소 : 각요소의 내용들이 위에서부터 아래로 쌓이는 형태

inline 요소 : 각요소의 내용들이 옆으로 자연스럽게 나열되는 형태

인라인 요소 내부에는 블럭요소가 들어갈수 없다.

html요소의 인라인과 블럭의 요소를 구분하고 싶다면 이곳을 이용해서 확인 인라인 인지 블럭인지 요소 확인 가능(보이는링크) 

문단을 의미하는 p요소와는 다르게 사용허는 형태 모두 그대로 유지하는 요소이지만,                                                   쓰임이 실제로는다소 불편함이 더 많은 기능이므로,사용률이적다.
문서의 제목을 사용할 경우에는 heading을 사용한다 <h1>~ <h6> 까지가 있다.  
엔티티 코드를 사용하여 꺽쇠(코드로인식하므로)및 기호의 표현을 코드로 기입하여 표기한다
